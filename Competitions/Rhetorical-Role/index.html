Benchmark for Understanding Indian Legal Documents

What is BUILD?

BUILD is a sequential sentence classification dataset which provides structure to Indian Court judgements using sentence rhetorical roles. This is created as part of OpenNyAI mission by EkStep Foundation, Thoughtworks , Aagami , National Law Schoolâ€™s law and technology society (Bangalore) and Rohini Nilekani Philanthropies.

For more details about data download, preprocessing, baseline model training, and evaluation please refer to GitHub repository.

Getting started

BUILD is distributed under a CC BY-SA 4.0 License. The training and development sets can be downloaded below.

Once you have built your model, you can use the evaluation script we provide below to evaluate model performance by running python evaluation.py <path_to_prediction> <path_to_gold>

To submit your models and evaluate them on the official test sets, please read our submission guide hosted on Codalab.
